# Easa Sageer Portfolio

A modern, responsive multi-page portfolio website built with HTML, CSS, and JavaScript.

## Pages
- `index.html` — Home
- `about.html` — About Me
- `skills.html` — Skills
- `education.html` — Education
- `projects.html` — Projects & Workshops
- `achievements.html` — Achievements & Interests
- `contact.html` — Contact

## Run locally
Open `index.html` in a browser. No build steps are required.

## Features
- Dark theme with blue/cyan accents and a light mode option
- Animated typing effect
- Smooth page transitions
- Animated stats and skill bars
- Glassmorphism cards
- Particle background on the home page
- AOS scroll animations
- Contact form interaction and download resume button
